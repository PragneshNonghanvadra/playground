# Labs: Mobile / Edge / Realtime

Tiny, focused experiments for the **Mobile / Edge / Realtime** topic. Each lab tests ONE concept.

## Conventions

- One subfolder per lab: `labs/mobile-edge-realtime/<lab-name>/`.
- Each lab's `README.md` follows [`templates/lab-template.md`](../../templates/lab-template.md).
- Every non-trivial lab includes a measurement and a failure experiment.
- Lessons feed back into [`docs/topics/mobile-edge-realtime/`](../../docs/topics/mobile-edge-realtime/).

## Subfolders

- `experiments/` — small experiments that don't yet warrant their own lab folder
- `notes/` — running notes from labs (rough, dated)
- `benchmarks/` — measurements and benchmark scripts
- `failure-experiments/` — chaos / fault-injection experiments
