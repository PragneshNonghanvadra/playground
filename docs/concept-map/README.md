# Concept Map

A living view of how engineering concepts connect across this repo.

## Files

- [`connections.md`](connections.md) — explicit cross-topic links.
- [`open-questions.md`](open-questions.md) — questions I haven't yet answered.
- [`learning-backlog.md`](learning-backlog.md) — what to study next, prioritized.

## Usage

- After each topic update, ask: "What does this connect to?" and update `connections.md`.
- After each lab, ask: "What did this _not_ answer?" and add to `open-questions.md`.
- Weekly, review `learning-backlog.md` and pick what to work on next.

The concept map is the **graph view** of this repo. The topic notes are the **node view**. Both should stay in sync.
