# Interviews

Interview-focused notes by area. Each subfolder collects:

- 30-second / 2-minute / deep-dive explanations of specific topics
- Practiced system-design problems (use [`templates/system-design-template.md`](../../templates/system-design-template.md))
- Common follow-up questions and strong-answer patterns

## Subfolders

- [`frontend/`](frontend/)
- [`backend/`](backend/)
- [`distributed-systems/`](distributed-systems/)
- [`system-design/`](system-design/)
- [`ai-engineering/`](ai-engineering/)
- [`product-engineering/`](product-engineering/)
- [`engineering-leadership/`](engineering-leadership/)

## Conventions

- File name: `kebab-case.md`, one topic per file.
- Use [`templates/interview-template.md`](../../templates/interview-template.md).
- Pair with the matching topic note in [`docs/topics/`](../topics/).
