# Tradeoff Analyses

Explicit tradeoff comparisons between competing approaches. Use [`templates/tradeoff-template.md`](../../../templates/tradeoff-template.md).

## When to write one

- A decision has 3+ viable options and the choice is context-dependent.
- A topic note's `tradeoffs.md` is too short to capture the full comparison.
- An ADR cites this tradeoff as background.
