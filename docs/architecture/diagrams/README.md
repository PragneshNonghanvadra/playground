# Diagrams

Saved diagrams for architecture, system design, and lab documentation.

## Conventions

- Prefer Mermaid (`.mmd`) or SVG. Avoid binary formats unless necessary.
- File name: `kebab-case.svg` or `kebab-case.mmd`, plus a sibling `kebab-case.md` describing what the diagram shows.
- Always link a diagram from at least one note. Orphan diagrams accumulate confusion.
