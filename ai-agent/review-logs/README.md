# Review Logs

Chronological log of AI-agent runs that touched this repo and what I changed.

## Conventions

- One file per non-trivial agent run: `YYYY-MM-DD-<short-slug>.md`.
- Use [`templates/ai-agent-review-template.md`](../../templates/ai-agent-review-template.md).
- After filling out the review, **also** copy or move it into the appropriate destination folder:
  - `accepted-suggestions/` for patterns I keep using
  - `rejected-suggestions/` for patterns I reject
  - `hallucinations/` for fabrications
  - `architecture-critiques/` for substantive design pushback

## Why this exists

To produce a longitudinal record of where AI helped, where it hurt, and how my prompts evolved. Without this, every agent session is amnesiac.
