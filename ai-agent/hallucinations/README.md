# Hallucinations

Fabrications worth remembering. Made-up APIs, invented config keys, nonexistent libraries, misattributed quotes, wrong defaults stated confidently.

## Format

One file per hallucination: `kebab-case-name.md`.

Include:

- The original prompt
- The hallucinated claim verbatim
- The truth, with a primary-source citation
- What this teaches me about _when_ the model hallucinates

## Why this exists

To recalibrate trust. The model is not uniformly unreliable — it's selectively unreliable, and the patterns are learnable.
